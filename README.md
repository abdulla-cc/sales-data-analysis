# Retail Sales Data Analysis

## Project Overview
Analysis of retail sales data to identify performance trends 
across regions, categories, and time periods.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts

## Key Insights
- West region has the highest sales ($710,219)
- Technology category is the most profitable
- Phones and Chairs are the top selling sub-categories
- Sales peaked around 2020
